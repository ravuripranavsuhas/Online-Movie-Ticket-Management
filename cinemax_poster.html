Project Based Learning from FWD
Abstract

The CineMax Online Movie Ticket Booking System is a web application developed using HTML, CSS, and JavaScript that simplifies the process of booking movie tickets. Users can view available movies, select showtimes, choose seats, and confirm bookings through a visually interactive interface. The platform eliminates long queues at theatres and provides a convenient digital solution for ticket reservations — demonstrating how front-end technologies can build a fully functional booking system with dynamic UI and booking logic.
